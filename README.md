            # practice_2.4 — GUI-приложения с внешними API

Графические приложения на Tkinter для работы с погодным API и API с фотографиями животных.

## Требования

- Python 3.10 или новее
- Tkinter (обычно входит в стандартную установку Python)
- Библиотеки: `requests`, `Pillow`

## Установка зависимостей

```bash
cd practice_2.4
python -m pip install -r requirements.txt
```

## Структура проекта

```
practice_2.4/
├── task_1.py          # Погода (OpenWeather API)
├── task_2.py          # Случайные фото котов и собак
├── requirements.txt
└── README.md
```

## Запуск

```bash
python task_1.py
python task_2.py
```

## Описание заданий

### task_1.py — Погода в городе

Приложение получает данные с [OpenWeather API](https://openweathermap.org/api) и показывает:

- температуру и «ощущаемую» температуру;
- описание погоды;
- влажность;
- иконку текущей погоды.

**Как получить API-ключ:**

1. Зарегистрируйтесь на https://openweathermap.org/
2. Перейдите в раздел API Keys
3. Скопируйте ключ и вставьте в поле **API-ключ** в приложении

Альтернатива — задать переменную окружения перед запуском:

```bash
# Windows PowerShell
$env:OPENWEATHER_API_KEY = "ваш_ключ"
python task_1.py
```

```bash
# Windows CMD
set OPENWEATHER_API_KEY=ваш_ключ
python task_1.py
```

**Использование:**

1. Введите API-ключ (если не задан через переменную окружения).
2. Введите название города (например, `Москва`, `Tomsk`).
3. Нажмите **«Показать погоду»**.

### task_2.py — Случайные фото котов и собак

Приложение использует бесплатные API:

- Коты: https://thecatapi.com/
- Собаки: https://dog.ceo/dog-api/

**Использование:**

1. Нажмите **«Получить кота»** — загрузится случайное фото кота.
2. Нажмите **«Получить собаку»** — загрузится случайное фото собаки.

API-ключ для этих сервисов не требуется.

## Возможные ошибки

| Ошибка | Решение |
|--------|---------|
| `Неверный API-ключ` (task_1) | Проверьте ключ на openweathermap.org; новый ключ может активироваться до 2 часов |
| `Город не найден` (task_1) | Проверьте написание города латиницей или кириллицей |
| Ошибка загрузки фото (task_2) | Проверьте подключение к интернету |
